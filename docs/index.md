## How to Use This Reference

This is not a textbook you read cover to cover. It is a **queryable knowledge base** designed for a practicing surgeon.

| Method | Best For |
|--------|----------|
| **AI Chatbot** | Ask any clinical, coding, or practice question in natural language. Every answer cites its source. |
| **Chapter Browse** | Deep-dive into a specific topic area with full evidence review. |
| **Search** | Find specific CPT codes, drug protocols, surgical techniques, or instrument specifications. |
| **Learning Graph** | Visualize concept relationships and prerequisite knowledge across all 20 chapters. |
| **Glossary** | Quick-reference definitions for 200+ OMS terms, instruments, and acronyms. |

## Design Principles

**Every answer is verifiable.** Clinical content cites AAOMS ParCare 2023 sections, peer-reviewed literature, FDA clearance numbers, and CPT/CDT/ICD-10 code references.

**Mobile-first.** Designed for phone access. The chatbot is the primary interface.

**Minimal images, maximum utility.** Images appear only for surgical anatomy, instrument identification, and radiographic interpretation.

**Living document.** Content is versioned and updated as guidelines, codes, and technology change.
